# Pawan Sai Kodali - AI Portfolio

This is a modern, interactive developer portfolio built with Next.js, featuring live AI demonstrations.

## Features

- **AI Playground**: Live interactive demos for Brain Tumor Detection, Hand Gesture Recognition, Crop Recommendation, and more.
- **AI Chatbot**: A custom assistant trained on portfolio data to answer recruiter questions.
- **Interactive 3D**: Hero section featuring Spline 3D animations.
- **Modern UI**: Built with ShadCN UI, Tailwind CSS, and Framer Motion.

## Getting Started

1. **Clone the repository**
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Set up Environment Variables**:
   Create a `.env` file in the root directory and add your Gemini API key:
   ```env
   GEMINI_API_KEY=your_actual_api_key_here
   ```
4. **Run the development server**:
   ```bash
   npm run dev
   ```

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS
- **Components**: ShadCN UI
- **AI Framework**: Google Genkit
- **LLM**: Google Gemini 2.0 Flash
- **Animations**: Framer Motion & Spline
