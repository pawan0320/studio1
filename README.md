# Pawan Sai Kodali - AI-Powered Developer Portfolio

Welcome to my interactive developer portfolio! This project is a comprehensive showcase of my expertise in Full-Stack Development, Artificial Intelligence, and IoT. Unlike a traditional resume, this portfolio features a live **AI Playground** where you can interact with production-grade AI models directly in your browser.

## 🚀 Key Features

### 🧠 AI Playground (Live Demos)
The heart of this portfolio is the interactive playground powered by **Google Gemini** and **Genkit**:
- **Brain Tumor Detection**: Upload MRI scans for AI classification (99.2% accuracy model logic).
- **Hand Gesture Recognition**: Real-time webcam analysis of hand states and movements.
- **AI Medical Assistant**: A symptom-checking bot providing preliminary health analysis and advice.
- **Crop Recommendation**: An agronomist bot that suggests optimal crops based on soil parameters.
- **Stock Price Prediction**: Conceptual LSTM-driven analysis for financial forecasting.
- **Drug Discovery**: Physicochemical analysis of molecules using SMILES strings.
- **Prosthetic Control**: A simulation of AI-driven control for robotic limbs.

### 🤖 Portfolio Chatbot
A custom-trained AI assistant is available site-wide to answer questions about my background, skills, and projects, providing a personalized experience for recruiters.

### 🎨 Modern UI/UX
- **Interactive 3D**: High-quality Spline animations in the Hero and Skills sections.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop using Tailwind CSS.
- **Dark Mode**: A sleek, developer-focused "Cyberpunk" aesthetic.

## 🛠️ Tech Stack

- **Framework**: [Next.js 15 (App Router)](https://nextjs.org/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **AI Framework**: [Google Genkit](https://github.com/google/genkit)
- **LLM**: [Google Gemini 2.0 Flash](https://deepmind.google/technologies/gemini/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Components**: [ShadCN UI](https://ui.shadcn.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/) & [Spline](https://spline.design/)

## 🛠️ Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/pawan0320/studio1.git
cd studio1
```

### 2. Install dependencies
```bash
npm install
```

### 3. Environment Variables
Create a `.env` file in the root directory and add your Gemini API key:
```env
GEMINI_API_KEY=your_actual_api_key_here
```

### 4. Run the development server
```bash
npm run dev
```
Open [http://localhost:9002](http://localhost:9002) to see the result.
